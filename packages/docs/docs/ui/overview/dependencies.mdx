`@trubittech/ui` is built using [Radix](https://radix-ui.com/) and [Stitches](https://stitches.dev) and is being built in the hopes of providing Stacks builders the design resources and components needed to quickly build prototypes. Most of the components are primitives with minimal styling intended to be used as <Highlight>a foundation for your UI</Highlight>.
